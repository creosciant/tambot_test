# TamBot - University AI Assistant

A simple, responsive chat interface prototype for a university-focused AI assistant.

## File Structure

- **main.html** - Main HTML file with all styling and modal templates
- **chat.js** - Chat functionality (message sending, typing indicators, responses)
- **auth-modal.js** - Login/signup modal popup functionality
- **auth-page.js** - Full-page authentication interface functionality
- **logo-modal.js** - Logo/image upload modal
- **tambot_logo.png** - AI avatar image

## Features

- Mobile-responsive design (tablet & mobile optimized)
- Greeting recognition system
- Dual authentication (modal popup + full page)
- University-focused suggestions and responses

## Theme Colors
